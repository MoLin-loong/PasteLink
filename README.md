# PasteLink

**PasteLink** – send text from your phone directly to your PC cursor.  
Both devices must be on the same local network. Scan the QR code shown on your PC with your phone, then type and paste instantly.  
Comes as a Python script and a standalone Windows `.exe` (no Python required). Zero app install on the phone.

**PasteLink（粘贴即连）**  
将手机上的文字直接发送到电脑光标所在位置。  
手机与电脑需处于同一局域网，用手机扫描电脑屏幕上的二维码即可连接，无需安装任何 App。  
提供 Python 原始脚本以及无需 Python 环境的 Windows 便携 `.exe` 文件。
